# WhatsApp Message Templates — Quick Reference

Submit these at: **Meta Business Suite → WhatsApp Manager → Message Templates → Create Template**

---

## 1. otp_verification (Authentication category — auto-approves)

- **Category:** Authentication
- **Name:** `otp_verification`
- **Language:** English
- **Body:**
  ```
  Your OneSpot verification code is: {{1}}. It expires in 5 minutes.
  ```

## How to Submit

1. Go to https://business.facebook.com
2. Select your WhatsApp Business Account
3. Navigate to **Message Templates**
4. Click **Create Template**
5. For each template above:
   - Select the category shown
   - Enter the name exactly as shown (lowercase, underscores)
   - Set language to English
   - Paste the body text
   - Submit for review
6. The `otp_verification` template auto-approves (authentication category)
7. Others typically approve within 1-24 hours

## Checking Status

Templates show status in the Message Templates dashboard:
- **Approved** — ready to use
- **Pending** — under Meta review
- **Rejected** — needs revision (check rejection reason)
